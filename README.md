# Micro robot arm (9g Micro Servo)

This is my attempt of building and programming my first ever circuit.
It's programmed using an Arduino Uno R3 board and 5 SG90 Micro Servomotors.

## Prerequesites

### Software:
- [GNU Make (4.4.1)](https://www.gnu.org/software/make/)
- [Arduino CLI (1.4.0)](https://docs.arduino.cc/arduino-cli/)

### Dependencies:
- Core:
    - arduino:avr (1.8.7)
- Libraries:
    - [MobaTools (v.2.7.0](https://docs.arduino.cc/libraries/mobatools/))

### Hardware:
- [1x Arduino Uno R3](https://docs.arduino.cc/hardware/uno-rev3/)
- [5x TowerPro SG90 micro servo (9g)](https://botland.com.pl/serwa-typu-micro/24406-towerpro-sg90-micro-serwomechanizm-analogowy-180-stopni.html)
- A bunch of cables, 2.5m and 3m screws
- ...and duck tape...
